## 🚀 部署指南

### 方式一：Vercel（推荐）

**预计时间**: 5分钟

```bash
# 1. 安装 Vercel CLI
npm install -g vercel

# 2. 登录
vercel login

# 3. 部署
cd my-writing-master
vercel

# 4. 设置环境变量
vercel env add NEXT_PUBLIC_CLAUDE_API_KEY

# 5. 生产部署
vercel --prod
```

### 方式二：本地运行

```bash
# 1. 安装依赖
npm install

# 2. 配置环境变量
cp .env.local.example .env.local
# 编辑 .env.local，添加 API Key

# 3. 启动开发服务器
npm run dev

# 4. 访问
http://localhost:3000
```

## 📱 使用流程

### 完整写作流程（约15-20分钟）

1. **审题** (2分钟): 分析题目，选择关键词
2. **寻宝** (3分钟): 挖掘素材，收集宝石
3. **搭建** (2分钟): 选择结构，搭建大纲
4. **写作** (8分钟): 完成初稿，至少300字
5. **润色** (2分钟): 对比优化
6. **登顶** (1分钟): 查看雷达图，庆祝

### 预期效果

- 学生能够独立完成一篇500-800字的记叙文
- 写作过程有明确的指导和反馈
- 通过游戏化提升写作兴趣
- 雷达图帮助了解自己的写作优势和不足
