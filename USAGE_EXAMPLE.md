# 使用示例

## 完整写作流程演示

### 场景：学生要写一篇作文《那一刻，我长大了》

---

## 步骤1: 审题大本营

**操作流程：**

1. 访问网站，点击"立即开始写作冒险"
2. 进入第一步页面
3. 在输入框中输入题目：**那一刻，我长大了**
4. 点击"让墨玉分析题目"

**墨玉的分析：**

```
关键词分析：
- "那一刻"：强调特定的时间点，需要有明确的场景描写
- "长大"：不仅是年龄增长，更是心理成熟、责任感增强

写作角度：
1. 挫折中成长：通过失败学会坚持
2. 责任中成长：承担家庭/社会责任
3. 理解中成长：理解父母的辛苦

素材线索：
- 回忆那个特定时刻的环境
- 当时的感受和心情变化
- 事件前后自己的变化
```

5. 选择关键词：**"长大"**
6. 点击"通关！"

**✅ 完成：** 理解了题目的核心要求

---

## 步骤2: 密林寻宝

**操作流程：**

1. 进入第二步页面
2. 在素材输入框中输入：

```
素材1：那天晚上，妈妈加班到很晚才回家，我给她倒了一杯热水。
```

3. 点击"添加素材"
4. 墨玉提出追问：
   - "那一刻你看到了妈妈的什么表情？"
   - "妈妈的手摸起来是什么感觉？"
   - "你当时心里在想什么？"

5. 回答追问，添加更多素材：

```
素材2：妈妈疲惫地靠在沙发上，眼角有了细纹，手很粗糙。
素材3：我突然意识到，妈妈为了我付出了这么多。
```

6. 继续挖掘，收集至少10个素材宝石

**✅ 完成：** 积累了丰富的写作素材

---

## 步骤3: 半山凉亭

**操作流程：**

1. 进入第三步页面
2. 选择结构模板：**欲扬先抑法**
3. 填写大纲：

```
抑-铺垫：描述自己以前不懂事，总是任性
转折点：看到妈妈加班回家疲惫的样子
扬-成长：主动照顾妈妈，意识到责任
升华：理解了成长的真正含义
```

4. 点击"大纲完成，前往写作！"

**✅ 完成：** 搭建了清晰的文章结构

---

## 步骤4: 绝壁攀岩

**操作流程：**

1. 进入第四步页面
2. 根据大纲开始写作：

```
那一刻，我长大了

以前的我，总是任性妄为。每次妈妈让我帮忙做家务，我总是找各种理由推脱。我觉得学习才是最重要的，其他的事情都应该由大人来做。

直到那个深夜...

（继续写作，至少300字）
```

3. 写作过程中：
   - 点击"问墨玉"获取帮助
   - 如果30秒无操作，触发"灵感阶梯"
   - 墨玉会提供写作建议

4. 完成初稿（至少300字）
5. 点击"初稿完成，前往润色！"

**⚠️ 注意：** 不要直接复制墨玉的建议，否则会触发"鹦鹉学舌陷阱"

---

## 步骤5: 云端魔镜

**操作流程：**

1. 进入第五步页面
2. 点击"智能润色"
3. 查看对比：

**原文：**
```
那天晚上，妈妈很累。
```

**润色版：**
```
那个深夜，妈妈拖着疲惫的身躯回到家，眼眶深陷，整个人仿佛被抽空了力气。
```

4. 可以手动修改润色版
5. 确认最终版本
6. 点击"润色完成，准备登顶！"

---

## 步骤6: 极顶插旗

**操作流程：**

1. 进入第六步页面
2. 享受庆祝动画 🎉
3. 查看金字拼图：**"破茧成蝶"**
4. 查看五维雷达图：
   - 立意：85分
   - 选材：90分
   - 结构：80分
   - 语言：75分
   - 情感：80分

5. 阅读自己的完整作品
6. 查看成就统计：
   - 总字数：520字
   - 关键词：3个
   - 素材数：12个
   - 关卡：6关

7. 可以选择：
   - "再写一篇"：重新开始
   - "分享成就"：分享到社交媒体

## 小贴士

### 1. 如何获得高分？

- 步骤1：选择最准确的关键词
- 步骤2：收集至少15个素材
- 步骤3：选择合适的结构模板
- 步骤4：字数控制在500-800字
- 步骤5：认真修改润色版

### 2. 如何避免抄袭？

- 理解墨玉的建议，用自己的话表达
- 不要直接复制示例句子
- 如果被检测到，按照提示改写

### 3. 如何提升写作能力？

- 多使用苏格拉底追问功能
- 学习高阶写法的技巧
- 对比原文和润色版的差异
- 多写多练

### 4. 如何保存作品？

- 浏览器会自动保存进度
- 可以在步骤6复制全文
- 建议截图保存雷达图
